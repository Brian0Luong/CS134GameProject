# DeepLock

## Overview
DeepLock is a single-player 3D puzzle platformer developed using the Unity Engine.  
The player navigates through rooms while solving puzzles to reach the exit.

As the player progresses, water gradually fills the room, creating time pressure.  
If the player fails to escape in time, their oxygen depletes, resulting in drowning.

---

## Gameplay Features
- Puzzle-solving across multiple rooms  
- Rising water system that increases difficulty  
- Oxygen/timer-based survival mechanic  
- Player movement and interaction system  
- Sound effects and animations for immersion
